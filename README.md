# رحلة كنز السيرة النبوية

تطبيق تفاعلي للأطفال لتعلّم السيرة النبوية بطريقة ممتعة ومغامرة.

## العرض المباشر

https://nadaazzzam.github.io/seerah-adventure/

## المحتوى

- رحلة تفاعلية عبر محطات السيرة النبوية
- واجهة عربية (RTL) مع دعم الوضع الفاتح والداكن
- مناسب للأطفال والتعليم في المنزل أو المدرسة

## حفظ البيانات

- **تلقائياً على الجهاز:** الطلاب، النجوم، الصور، وتقدم اللعبة تُحفظ في المتصفح.
- **مزامنة سحابية (اختياري):** اتبع [`BACKEND.md`](BACKEND.md) لربط Supabase مجاني ومشاركة نفس البيانات بين الأجهزة.
